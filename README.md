# Implementaci-n-de-Telefon-a-IP-Wifi-con-Matrix-UCS-y-Cisco-Meraki
Implementación de una infraestructura de telefonía IP wifi entre sedes utilizando Matrix UCS, Cisco Meraki, segmentación por VLAN, IPs estaticas y dispositivos Grandstream, se incluira el diseño de la topología, direccionamiento IP y pruebas de conectividad.
