# Implementación-de-Telefonía-IP-Wifi-con-Matrix-UCS-y-Cisco-Meraki

En este proyetco se realizara la implementación de una infraestructura de telefonía IP wifi entre sedes en entorno de una infraestructura OT (Subestaciones Electricas) usando las tecnologias de Matrix UCS, Cisco Meraki, Telefonos GrandStream WIFI.

El laboratorio incluye la segmentación mediante VLAN, configuración de red, integración de dispositivos Grandstream y pruebas de conectividad.

> 🚧 Proyecto en desarrollo y se irán agregando nuevas funcionalidades y documentación conforme avance la implementación.


## 🎯 Objetivos

- Implementar una solución de telefonía IP entre sedes.
- Segmentacion de la red mediante VLAN.
- Uso de direccionamiento Estatico.
- Integrar dispositivos Grandstream con Matrix UCS.
- Administrar el Access Point mediante Cisco Meraki Dashboard.
- Validar la comunicación entre todos los dispositivos.

## 🛠 Tecnologías utilizadas

- APs Cisco Meraki
- Matrix UCS en Alta disponibilidad HA
- Grandstream WP836
- Telefonía IP Fijo VP510
- Switch Cisco C1300
